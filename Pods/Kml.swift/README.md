# KML.swift

Simple KML parser for Swift.  

## Libraries

- [AEXML](https://github.com/tadija/AEXML) via cocoapods
- [UIColor-Hex-Swift](https://github.com/yeahdongcn/UIColor-Hex-Swift) includes

## Supporting TAGs

- Style
- StyleMap
- PolyStyle
- LineStyle
- MultiGeometry
- Polygon
- LineString
- Point
- Folder
- Placemark
